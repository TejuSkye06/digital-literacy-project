# digital-literacy-project
This project consist of 5 tasks altogether. It included creating an infographic on digital literacy, building a professional online portfolio, exploring coding and collaboration platforms, drafting formal emails with a social media etiquette checklist, and studying cybercrime through a case study and prevention guide.

## Task 1: Digital Literacy Infographic
A one-page infographic was created using Canva to explain the concept of digital literacy, its importance, safe internet practices, and maintaining a professional online presence. The design uses clear visuals and concise points to ensure easy understanding.
